# Bienvenue au Gestionnaire de Catalogue Produit

## Objectif du Projet
Le **Service de Gestion du Catalogue Produit** est une application backend robuste conçue pour gérer les opérations principales d'un système de catalogue. Elle est construite pour être évolutive, maintenable et facile à étendre.

Les objectifs principaux de ce projet sont :

- **Gérer les Produits** : Créer, mettre à jour et récupérer les informations des produits.  
- **Traiter les Commandes** : Gérer les commandes clients avec validations et règles métier.  
- **Évolution Future** : Prêt pour intégrer les modules de gestion des Clients et des Fournisseurs.  

## Fonctionnalités Clés

- **Clean Architecture** : Séparation des responsabilités pour une meilleure maintenabilité.  
- **Domain-Driven Design** : La logique métier est encapsulée dans la couche Domaine.  
- **API REST** : Expose les fonctionnalités via des endpoints HTTP standards.
